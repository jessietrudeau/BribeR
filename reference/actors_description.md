# Actor Descriptions

Short biographical descriptions for a subset of individuals who appear
in the Vladivideos transcripts.

## Usage

``` r
actors_description
```

## Format

A tibble with 79 rows and 2 variables:

- speaker_std:

  Standardized speaker identifier, matching `speaker_std` in the
  transcripts corpus.

- description:

  Brief description of the individual's role.

## Source

Manually compiled from the Vladivideos archive and related published
research.
